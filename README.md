# snap-ui [![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

⚡ A curated collection of snap-ready UI components — HTML/CSS, JavaScript widgets, PHP/CI snippets &amp; animations. Clean. Fast. Production-ready.

> Snap it. Ship it. A professional collection of ready-to-use UI components,
> JavaScript widgets, PHP/CodeIgniter snippets, and animations — built for
> developers who value clean code and fast delivery.

---

## ✨ What's Inside

| Category                | Description                                   |
| ----------------------- | --------------------------------------------- |
| 🎨 HTML/CSS Components  | Buttons, cards, modals, navbars, forms & more |
| ⚙️ JavaScript Widgets   | Dropdowns, tabs, sliders, toast notifications |
| 🐘 PHP/CI Snippets      | CodeIgniter helpers, models, auth patterns    |
| 🎬 Animations & Effects | CSS/JS transitions, loaders, hover effects    |

---

## 📁 Folder Structure

```
snap-ui/
├── html-css/         # UI components & layouts
├── js-widgets/       # JavaScript UI widgets
├── php-ci/           # PHP & CodeIgniter snippets
├── animations/       # CSS & JS animations
└── README.md
```

---

## 🚀 How to Use

1. Browse to the category folder you need
2. Each project has its own folder with a `README.md`
3. Copy the snippet or open `index.html` directly in browser
4. No build tools required — just snap and ship!
